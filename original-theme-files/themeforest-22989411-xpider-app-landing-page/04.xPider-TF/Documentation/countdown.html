Countdown

A simple and HTML agnostic date countdown plugin which is very easy to set the desire time.

Example

  • Days
  • Hours
  • Minutes
  • Seconds 

                                        <ul class="countdown d-flex mb-40 justify-content-center justify-content-lg-start" id="date" data-date="Sep 30, 2019">
                                <li class="d-flex flex-column mr-60 text-left">
                                    <span class="countdown__count h3-font font-w-700 color-primary" id="days"></span>
                                    <span>Days</span>
                                </li>
    
                                <li class="d-flex flex-column mr-60 text-left">
                                    <span class="countdown__count h3-font font-w-700 color-primary" id="hours"></span>
                                    <span>Hours</span>
                                </li>
    
                                <li class="d-flex flex-column mr-60 text-left">
                                    <span class="countdown__count h3-font font-w-700 color-primary" id="minutes"></span>
                                    <span>Minutes</span>
                                </li>
    
                                <li class="d-flex flex-column text-left">
                                    <span class="countdown__count h3-font font-w-700 color-primary" id="seconds"></span>
                                    <span>Seconds</span>
                                </li>
                            </ul>

Updating the time

Change the value from data-date="" to update the time
